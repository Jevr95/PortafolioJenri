# JENRI · Always Up
### Portfolio Personal — José Enrique Villarreal Rubio

> *Ingeniero de formación, desarrollador web por convicción.*

Landing page portfolio desarrollada con HTML, CSS y JavaScript vanilla. Diseño editorial, secciones animadas y despliegue estático sin dependencias de frameworks.

---

## Demo en vivo

**[Ver portfolio →](https://Jevr95.github.io/REPO)**

---

## Características

- **Hero** con nombre, descripción y foto
- **Sobre mí** con trayectoria, estadísticas y botón para ver CV en modal
- **Filosofía de trabajo** — 3 tarjetas con efecto flip (hover / tap)
- **Proyectos** — grid con imagen de fondo, gradiente y enlaces a sitios en vivo
- **Skills** — grid 2×2 con tags diferenciados por nivel
- **Contacto** — formulario + redes sociales
- Scroll reveal con IntersectionObserver
- Mascota interactiva con easter egg
- Diseño responsive (mobile-first)
- Sin frameworks, sin dependencias externas

## Uso local

```bash
# Clona el repositorio
git clone https://github.com/Jevr95/REPO.git

# Entra a la carpeta
cd REPO

# Ábrelo con Live Server en VS Code
# o directamente con doble clic en index.html
```

> **Nota:** el botón *Ver mi CV* requiere que `EnriqueVillarreal_CV_CH65_R2.pdf` esté en la misma carpeta que `index.html`. No incluir el PDF en el repositorio si es información sensible — puedes ignorarlo con `.gitignore`.
